## How to Capture

### Using the automated script (recommended):

```bash
# Prerequisites: Android emulator running + Expo dev server
npx expo start --android

# In another terminal:
./scripts/capture-screenshots.sh

# For iOS only:
./scripts/capture-screenshots.sh --ios-only

# Clean previous output:
./scripts/capture-screenshots.sh --clean
```

### Manual capture with Maestro:

```bash
# Start the app on your device/emulator
# Then run the screenshot flow:
maestro test .maestro/flows/screenshots/capture.yaml

# Or for a single screenshot during development:
maestro studio
# (then use the takeScreenshot command interactively)
```

## Multi-Language Screenshots

For localized App Store listings, run the capture script with different
emulator locales:

```bash
# Hindi (India)
adb shell setprop persist.sys.locale hi-IN
./scripts/capture-screenshots.sh

# English (US)
adb shell setprop persist.sys.locale en-US
./scripts/capture-screenshots.sh
```

## Device Frames

After capturing raw screenshots, use a tool like **Figma** or **AppScreens**
to wrap them in device frames before uploading to App Store Connect.
